# Video Ad Script Prompt

Title: Video Ad Script Prompt
Category: video-generation
Use case: Ad script
Version: v1
Tags: video-generation, ad-script, marketing, short-video, conversion

## Prompt

Write a short video ad script for `{product_name}` promoting `{offer}` to `{target_audience}`. The script should fit `{duration}` and use `{tone}`. Start with a strong hook, show the problem, introduce the product, demonstrate the benefit, and end with `{call_to_action}`.

Include scene-by-scene direction, voiceover, on-screen text, and visual notes. Keep the language specific, concrete, and easy to film or generate.

## Input variables

- `product_name`: Product or brand name.
- `offer`: Main offer or campaign message.
- `target_audience`: Intended viewers.
- `duration`: Video length.
- `tone`: Practical, funny, premium, urgent, calm, etc.
- `call_to_action`: Final action.
- `platform`: TikTok, Reels, YouTube Shorts, LinkedIn, etc.

## Expected output

A timed video ad script with hook, scenes, voiceover, on-screen text, visual direction, and CTA.

## Example input

`product_name`: PromptVault  
`offer`: reusable prompt folders for creative teams  
`target_audience`: marketing managers  
`duration`: 30 seconds  
`tone`: crisp and practical  
`call_to_action`: start a free workspace  
`platform`: LinkedIn

## Example output

A 30-second LinkedIn ad script opening with "Your best prompts should not live in yesterday's chat." Scenes show scattered docs, a clean shared prompt library, folder organization, one-click copy, and a CTA to start a free workspace.
