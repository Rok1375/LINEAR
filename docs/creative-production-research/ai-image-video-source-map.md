## Recommended Prototype Shortlist

1. **OpenAI Images API**: cleanest first hosted image generation/editing implementation target.
2. **Google Vertex AI Imagen + Veo**: enterprise-grade source for image/video plus watermark and safety controls.
3. **Adobe Firefly API**: production creative-suite option with image, video, expand, and fill workflows.
4. **Runway + Luma**: practical hosted video generation/API candidates.
5. **Diffusers + Hugging Face Hub**: local/open-source baseline for image, video, control, LoRA, and upscaling experiments.
6. **ComfyUI workflow docs + registry + templates**: primary local workflow automation path with inspectable JSON metadata.
7. **Replicate + fal**: fast hosted model evaluation layer for comparing many image/video models before committing.

## Manual Review Flags

- **Community hubs**: Civitai and Hugging Face require per-model license, safety, and provenance checks.
- **Product-only sources**: Midjourney and some Krea/Firefly surfaces are valuable for prompt/control research but may not expose crawlable workflow metadata.
- **Community discussions, tool directories, and YouTube tutorials**: useful for workflow discovery, but should be treated as leads only until reproduced in ComfyUI, Diffusers, or a hosted API.
- **Custom ComfyUI nodes**: inspect code, registry entry, dependencies, and security posture before installing; never trust a workflow import blindly.
